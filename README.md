# Asia Family Trip 2026
This repository contains the source code for a static website created to share
all key information about our family trip to Asia (April–May 2026).
The website is intended for non-technical users and focuses on clarity,
accessibility, and ease of use across devices and countries.

## Trip Overview
- **Dates:** April 10 – May 2, 2026
- **Countries:**
  - Vietnam (Hanoi, Da Nang, Hoi An)
  - Thailand (Bangkok, Ayutthaya)
  - Laos (Vientiane, Luang Prabang)
- **Participants:** Family members traveling from France and the Philippines

## Website Purpose
The site provides:
- A global overview of the trip
- Detailed itinerary by country and city
- Practical travel tips
- Budget information
- Shared agenda and key dates
- Useful links and documents

## Technical Stack
- HTML5
- CSS3
- Vanilla JavaScript
- No backend
- No build step
